# Backend Engineering Curriculum

This curriculum covers a complete Go backend engineering course from foundations through a capstone project.

## Modules

1. [Module 1: Foundations of Backend Engineering](module-01-foundations.md)
   - Introduction to backend systems, architecture, APIs, security, and team workflows.
2. [Module 2: Go Fundamentals and Core Programming](module-02-go-fundamentals.md)
   - Go language fundamentals, concurrency, and building reliable services.
3. [Module 3: Data, Testing, and Persistence](module-03-data-testing-and-persistence.md)
   - JSON, validation, persistence, testing, error handling, and debugging.
4. [Module 4: Databases and HTTP APIs](module-04-databases-and-http-apis.md)
   - Relational data modeling, REST API design, authentication, and backend services.
5. [Module 5: Advanced Backend Architecture and Scaling](module-05-advanced-backend-architecture-and-scaling.md)
   - Scalable architecture, deployment patterns, observability, security, and performance.
6. [Module 6: Production Backend Engineering and Professional Practices](module-06-production-backend-engineering-and-professional-practices.md)
   - CI/CD, code review, release management, security, incident response, and professional practices.
7. [Module 7: Capstone Project and Engineering Showcase](module-07-capstone-project-and-engineering-showcase.md)
   - Capstone planning, architecture, testing, deployment, demos, and retrospectives.

## How to Use This Curriculum

- Read each module in sequence.
- Complete the exercises, assignments, and mini-projects in each module.
- Use the module assignments to build practical artifacts and documentation.
- Finish with Module 7 by completing a capstone backend project and reflective retrospective.
